Formation Rules

  1. Every sentence letter is a formula.
  2. If φ is a formula, so is ¬φ.
  3. If φ and ψ are formulas, so are (φ ∧ ψ), (φ ∨ ψ), (φ → ψ).
  4. Nothing else is a formula.

The outer parentheses of the main connective may be omitted.
