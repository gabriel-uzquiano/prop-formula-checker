Propositional Logic — Syntax

Vocabulary

  • Sentence lettersp, q, r, s, t (with optional numeric subscripts)
  • Connectives¬   ∧   ∨   →
  • Parentheses(   )

Formation rules

  1. Every sentence letter is a formula.
  2. If φ and ψ are formulas, so are:
    ¬φ,   (φ ∧ ψ),   (φ ∨ ψ),   (φ → ψ)
  3. Nothing else is a formula.

The outer parentheses of a formula may be omitted when it is not part of a larger formula.

Examples

ExpressionStatus
(p ∨ (q → ¬r))official formula
¬(p ∧ q)official formula
p → qunofficial formula (abbreviates (p → q))
¬(p → A)not a formula (A is not a sentence letter)
